## Concept
---
오늘 뭐보지? -> 볼만한 영화 없나?
유저들이 영화를 추천하고, 유저들은 추천된 영화 중 본 영화는 제외하고 추천 영화 리스트를 받음. ( 최신 추천순 , 장르 추천순 )
점수제 대신 3단계로 평가함. (추천해요. / 잘모르겠어요. / 별로예요.)
추천해요에 추천된 영화는 최신 추천순에 반영되어 나타날수 있음.
